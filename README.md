## Git turtorial
```
- nếu đã có nhánh đang làm dở thì phải checkout sang nhánh đó trước : git checkout <tên nhánh>
nếu k có thì: 
git checkout -b <tên nhánh mới>
Làm xong thì : 
git add <tên thư mục>
git pull origin main
git push origin <tên nhánh đang làm/checkout>
